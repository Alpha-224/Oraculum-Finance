# 📱 Project Workflow & Repo Rules

### 📂 Designated Territories
- **Logic Lead (Me):** Works exclusively in `/backend`.
- **Frontend Team:** Works exclusively in `/frontend/lib`.
- **Data Team:** Works exclusively in `/data_layer`.

### 🛠 Git Rules
- **Main is Protected:** Never push directly to `main`.
- **Branches:** Use `feature/feature-name`.
- **Syncing:** Run `git pull origin main` and `cd frontend && flutter pub get` daily.
